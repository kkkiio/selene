# Validation fixtures

These fixtures were written from scratch for Selene XAML. They are inspired by
the categories exercised by NoesisGUI's public samples (layout, text, images,
composition, data, input, and animation), but contain no copied Noesis markup,
text, names, or assets.

`component-card.xaml` is compiled and registered before
`component-card-usage.xaml`. All other files use only the built-in Selene UI
schema.
