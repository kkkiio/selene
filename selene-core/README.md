# KKKIIO/selene

Core ECS/runtime package for Selene.

It includes app/runtime orchestration, ECS data flow, ordinary assets, named
texture-atlas import/resolution, scenes, animation, 2D/3D rendering, UI,
compiled View hosting, physics, and editor-bridge document codecs.

## Install

```bash
moon add KKKIIO/selene
```

## Backends

Use this package together with:

- `KKKIIO/selene_webgpu` for web builds
- `KKKIIO/selene_raylib` for native builds
